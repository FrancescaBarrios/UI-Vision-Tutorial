Automatizaciones con UI.Vision

Una guía paso a paso para programadores junior

¿Qué es UI.Vision?

UI.Vision RPA es una herramienta gratuita de automatización visual para navegadores que permite realizar tareas como rellenar formularios, navegar por páginas web, hacer scraping, y más. Ideal para tareas repetitivas y pruebas web.

Comandos Básicos

  • open: Abre una URL específica.
  • click: Hace clic sobre un elemento identificado.
  • type: Escribe texto en un campo de entrada.
  • store: Guarda un valor o variable.
  • echo: Imprime un mensaje en la consola de UI.Vision.

Comandos Intermedios

  • storeText: Extrae y guarda el texto de un elemento.
  • storeValue: Guarda el valor de un campo de formulario.
  • if_v2 / endif: Estructura condicional.
  • verifyElementPresent: Verifica que un elemento existe.
  • gotoLabel / label: Control de flujo personalizado.

Comandos Avanzados

  • csvRead / csvReadArray: Leer datos desde archivos CSV para automatización masiva.
  • executeScript / runScript: Ejecuta JavaScript personalizado.
  • storeEval: Evalúa expresiones JavaScript y almacena resultados.
  • times: Repite un bloque de comandos.
  • screenshot / captureEntirePageScreenshot: Toma capturas de pantalla.
